# College Events Manager System

A full-stack web application for managing college events, built as part of a DBMS group project.
The system allows administrators to create and manage events, while students can view and register for events.  
It follows a clean separation of concerns between frontend, backend, and database layers.

## Tech Stack

### Frontend
- React (Vite)

### Backend
- Node.js
- Express.js
- RESTful APIs
- CORS enabled

### Database
- Oracle Database
- SQL for data storage and retrieval

## Features
- View list of college events
- Backend REST APIs for event management
- Database connectivity using OracleDB
